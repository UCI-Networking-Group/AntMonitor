edu.uci.calit2.antmonitor.lib.logging

Class FileUploader

  • java.lang.Object
    • edu.uci.calit2.antmonitor.lib.logging.FileUploader

  • public class FileUploader
extends java.lang.Object
    Service for uploading network traffic log files to the remote data store. Uploads are handled on a single worker thread in FIFO manner (see IntentService documentation). Due to the fact that this Service is implemented as an intent service, we create a new connection for each file to upload. For each request, a "Connection : close" header is added to effectively ensure that each connection is closed, as it is not going to be re-used.
