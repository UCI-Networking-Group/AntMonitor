Class IpDatagram

  • java.lang.Object
    • edu.uci.calit2.antmonitor.lib.util.IpDatagram

  • public class IpDatagram
extends java.lang.Object
    This class represents an IP Datagram Packet and supplies various of static methods to parse IP datagrams. Note that this class only supports IPv4. There is no guarantee that any of these methods will work with IPv6 datagrams.
