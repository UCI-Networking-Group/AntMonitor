edu.uci.calit2.antmonitor.lib.logging

Class PacketConsumer

  • java.lang.Object
    • edu.uci.calit2.antmonitor.lib.logging.PacketConsumer

  • public class PacketConsumer
extends java.lang.Object
    This class receives packets from PacketQueueReader via the consumePacket call. Children of this class can override consumePacket and process the packets as they wish. Children of this class should be passed to the system during the connect call of VpnController so that PacketQueueReaders know to use them instead of the default PacketConsumer.
